# BRIEF KOMPONEN: ErrorState
# Project: YUBASHOP
# Lokasi: resources/js/Components/Shared/ErrorState.tsx
# Tech: Tailwind CSS + lucide-react
# Tipe: React component

---

## DESKRIPSI

Komponen yang ditampilkan ketika terjadi error rendering, error data loading, atau kegagalan aksi backend. Menampilkan status kesalahan secara visual dan memberikan tombol aksi coba lagi.

---

## PROPS INTERFACE

```typescript
interface ErrorStateProps {
  message?: string        // default: "Gagal memuat data. Silakan coba lagi."
  onRetry?: () => void    // callback tombol "Coba Lagi"
}
```

---

## TAMPILAN

- Layout: flex-col items-center justify-center, min-h-[300px]
- Ikon: TriangleAlert dari lucide-react, w-16 h-16, text-red-500
- Title: "Terjadi Kesalahan", text-lg font-semibold text-slate-300, mt-4
- Pesan: dari props `message` atau default, text-sm text-slate-500, mt-2
- Tombol "Coba Lagi" (jika onRetry ada):
  - mt-6, bg-red-600 hover:bg-red-700
  - Ikon RotateCcw di kiri teks
  - Rounded, px-4 py-2

---

## CONTOH PEMAKAIAN (Inline handling)

```tsx
import { ErrorState } from '@/Components/Shared/ErrorState'
import { useState } from 'react'

const [errorMsg, setErrorMsg] = useState<string | null>(null)

function handleFetchData() {
  setErrorMsg(null)
  // Logic fetch data...
}

if (errorMsg) {
  return <ErrorState message={errorMsg} onRetry={handleFetchData} />
}
```

---

## CATATAN

- JANGAN tampilkan error teknis raw database atau stack trace PHP ke user storefront/admin.
- Pesan error harus ramah, ringkas, berbahasa Indonesia, dan actionable (memberi tahu apa yang harus dilakukan pembeli/admin).
- Desain default dark mode.
