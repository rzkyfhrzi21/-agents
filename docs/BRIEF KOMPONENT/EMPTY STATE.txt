# BRIEF KOMPONEN: EmptyState
# Project: YUBASHOP
# Lokasi: resources/js/Components/Shared/EmptyState.tsx
# Tech: Tailwind CSS + lucide-react
# Tipe: React component

---

## DESKRIPSI

Komponen yang ditampilkan ketika fetch data berhasil tapi hasilnya KOSONG. Menggantikan tabel/list kosong dengan tampilan yang informatif dan ramah.

---

## PROPS INTERFACE

```typescript
interface EmptyStateProps {
  icon?: React.ReactNode         // default: <Database /> dari lucide-react
  title: string                  // contoh: "Belum Ada Game"
  description?: string           // contoh: "Tambahkan game pertama untuk mulai."
  action?: {
    label: string                // contoh: "Tambah Game"
    href?: string                // link navigasi (menggunakan Inertia Link)
    onClick?: () => void         // atau callback function
  }
}
```

---

## TAMPILAN

- Layout: flex-col items-center justify-center, min-h-[300px]
- Ikon: w-16 h-16, text-slate-500
- Title: text-lg font-semibold text-slate-300, mt-4
- Description: text-sm text-slate-500, mt-2
- Tombol aksi (opsional):
  - mt-6, bg-emerald-600 hover:bg-emerald-700
  - Rounded, px-4 py-2
  - Jika ada `href` → render sebagai `<Link>` dari `@inertiajs/react`
  - Jika ada `onClick` → render sebagai `<button>`

---

## CONTOH PEMAKAIAN

```tsx
// Di dalam file React View Page setelah menerima props data kosong
import { EmptyState } from '@/Components/Shared/EmptyState'

if (games.length === 0) {
  return (
    <EmptyState
      title="Belum Ada Game"
      description="Tambahkan game pertama untuk mulai berjualan."
      action={{ label: "Tambah Game", href: route('admin.games.create') }}
    />
  )
}
```

---

## CATATAN

- Tampilan default dark mode mengikuti tema panel admin.
- Tombol aksi bersifat opsional.
- Link internal WAJIB menggunakan `<Link href={...}>` dari `@inertiajs/react` agar transisi antar halaman tetap single page app (SPA) tanpa full page reload.
