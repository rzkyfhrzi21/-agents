# Project: YUBASHOP

# Tech: Tailwind CSS + lucide-react
# Tipe: React component

---

## DESKRIPSI

Komponen skeleton loading reusable yang ditampilkan saat data sedang diambil atau diproses di backend. Menggantikan pemakaian spinner tradisional dengan visual yang menyerupai bentuk konten asli (tabel, card, statistik).

---

## PROPS INTERFACE

```typescript
interface LoadingSkeletonProps {
  rows?: number       // default: 5
  columns?: number    // default: 4
  type?: 'table' | 'card' | 'stats'  // default: 'table'
}
```

---

## TAMPILAN PER TYPE

### 1. type: 'table' (default)
- Simulasi baris-baris tabel yang memudar (shimmer effect).
- Header: 1 bar abu-abu gelap panjang penuh (`bg-slate-700`).
- Body: `rows` baris × `columns` kolom, setiap sel = bar abu shimmer dengan lebar bervariasi (60%-90%) agar terlihat alami.

### 2. type: 'card'
- Grid kartu: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-6`.
- Setiap kartu memiliki tinggi `h-48`, berwarna `bg-slate-800` dengan simulasi thumbnail di atas dan baris teks di bawah.

### 3. type: 'stats'
- Baris kartu statistik ringkas: `grid-cols-2 sm:grid-cols-4 gap-4`.
- Setiap kartu memiliki tinggi `h-24` dengan placeholder ikon bulat dan teks memudar.

---

## EFEK SHIMMER & SINKRONISASI

- Menggunakan utilitas bawaan Tailwind `animate-pulse` untuk animasi denyut memudar.
- Background dasar: `bg-slate-700/50`.

---

## PEMAKAIAN DALAM COMPONENT PAGE (Inertia React)

Karena Inertia memuat seluruh halaman secara asinkronus, loading indicator global (top progress bar) diatur otomatis di `app.tsx`. Namun untuk aksi fetch data inline (asinkronus) atau ketika memuat data parsial, gunakan skeleton di tingkat komponen:

```tsx
import { LoadingSkeleton } from '@/Components/Shared/LoadingSkeleton'
import { useState } from 'react'

export default function GamesIndex() {
  const [loading, setLoading] = useState(false)
  const [games, setGames] = useState([])

  return (
    <div className="space-y-6">
      {loading ? (
        <LoadingSkeleton type="table" rows={6} columns={4} />
      ) : (
        <GameTable data={games} />
      )}
    </div>
  )
}
```
