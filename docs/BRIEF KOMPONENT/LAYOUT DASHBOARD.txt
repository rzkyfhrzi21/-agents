# BRIEF KOMPONEN: Layout Dashboard Admin
# Project: YUBASHOP
# Lokasi: resources/js/Components/Admin/Layout.tsx
# Tech: Laravel 12 + Inertia React + Tailwind CSS + lucide-react
# Tipe: React Layout Component

---

## DESKRIPSI

Layout utama halaman admin dashboard YUBASHOP. Terdiri dari Sidebar navigasi kiri dan Header atas. Default styling menggunakan dark mode untuk visual premium.

---

## STRUKTUR DIREKTORI LAYOUT

```
resources/js/Components/Admin/
├── Layout.tsx                       # Wrapper utama layout (sidebar + header + content)
├── Sidebar.tsx                      # Panel navigasi kiri (responsive drawer di mobile)
├── Header.tsx                       # Bar bagian atas (user info, dropdown logout, hamburger)
└── SidebarItem.tsx                  # Menu item navigasi
```

## HEADER

Tinggi: `h-16`, full width.

- **Kiri:** Hamburger button (hanya mobile) untuk membuka/tutup drawer sidebar + judul halaman.
- **Kanan:** Nama user login + avatar + dropdown menu (Profil, Logout).
- **Proses Logout:** Tombol logout menembak POST request ke `/logout` via router Inertia:
  ```tsx
  import { router } from '@inertiajs/react'
  
  function handleLogout() {
    router.post(route('logout'))
  }
  ```

---

## KONTEN AREA
- Padding: `p-6`
- Background: `bg-slate-900`
- Scroll independen, tinggi minimal layar (`min-h-screen`)

---

## MOBILE BEHAVIOR (< 768px)
- Sidebar otomatis berubah menjadi drawer dengan background gelap overlay (`bg-black/50`).
- Hamburger menu di header membuka drawer dari sisi kiri ke kanan (`translate-x-0`).
- Mengklik area luar drawer (overlay) atau menu item otomatis menutup sidebar.

---

## DETEKSI MENU AKTIF

Mendeteksi menu aktif menggunakan URL saat ini yang diperoleh dari Inertia `usePage`:

```tsx
import { usePage } from '@inertiajs/react'

const { url } = usePage()
const isActive = url.startsWith('/admin/games')
```
- Menu aktif disorot dengan: `bg-emerald-600/20 text-emerald-400 border-l-2 border-emerald-500`
- Menu tidak aktif: `text-slate-400 hover:bg-slate-800 hover:text-slate-200`
