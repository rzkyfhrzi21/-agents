# Project: YUBASHOP

# Tech: @tanstack/react-table + shadcn/ui + lucide-react + Tailwind CSS
# Tipe: React component

---

## DESKRIPSI

Komponen DataTable generik yang digunakan di SELURUH halaman admin dashboard untuk menampilkan data dalam bentuk tabel. Dibangun di atas @tanstack/react-table dan shadcn/ui Table component.

---

## PROPS INTERFACE

```typescript
import { ColumnDef } from '@tanstack/react-table'

interface DataTableProps<TData, TValue> {
  columns: ColumnDef<TData, TValue>[]
  data: TData[]
  isLoading?: boolean          // default: false
  searchKey?: string           // kolom yang bisa dicari (client-side)
  searchPlaceholder?: string   // placeholder input search
  onEdit?: (row: TData) => void
  onDelete?: (row: TData) => void
  onView?: (row: TData) => void
  // Server-side pagination
  pageCount?: number
  pageIndex?: number
  pageSize?: number
  onPaginationChange?: (pageIndex: number, pageSize: number) => void
}
```

---

## FITUR WAJIB

1. **COLUMN DEFINITION (per halaman):**
   - Setiap halaman mendefinisikan kolom sendiri di file `Columns.tsx`
   - Support custom render via ColumnDef `cell` property (seperti badge status, format mata uang, thumbnail)

2. **SEARCH / FILTER (Client-side):**
   - Input search di atas tabel, memfilter berdasarkan `searchKey` yang diberikan dengan debounce 300ms.

3. **PAGINATION:**
   - Menggunakan shadcn Pagination component.
   - Support server-side pagination dengan info: "Menampilkan X dari Y data".
   - Default per_page: 10 (sesuai Laravel $perPage default).

4. **KOLOM AKSI:**
   - Ikon Eye (biru) untuk Lihat, Pencil (hijau) untuk Edit, Trash2 (merah) untuk Hapus.

5. **STATE HANDLING:**
   - **LOADING:** Tampilkan skeleton shimmer (5 baris, sesuai jumlah kolom).
   - **EMPTY:** Tampilkan EmptyState component.
   - **DATA:** Render tabel normal.

## CONTOH PEMAKAIAN

```tsx
// resources/js/Pages/Admin/Games/Columns.tsx
import { ColumnDef } from '@tanstack/react-table'
import { Badge } from '@/Components/UI/badge'

export const columns: ColumnDef<Game>[] = [
  { accessorKey: 'name', header: 'Nama Game' },
  { accessorKey: 'sub_name', header: 'Developer' },
  {
    accessorKey: 'is_popular',
    header: 'Populer',
    cell: ({ row }) => (
      <Badge variant={row.original.is_popular ? 'default' : 'secondary'}>
        {row.original.is_popular ? 'Ya' : 'Tidak'}
      </Badge>
    ),
  },
]

// resources/js/Pages/Admin/Games/Index.tsx
import { router } from '@inertiajs/react'
import { DataTable } from '@/Components/Shared/DataTable'
import { columns } from './Columns'

export default function GamesIndex({ games }) {
  return (
    <DataTable
      columns={columns}
      data={games.data}
      searchKey="name"
      searchPlaceholder="Cari game..."
      onEdit={(game) => router.get(route('admin.games.edit', game.id))}
      onDelete={(game) => openDeleteModal(game)}
      pageCount={games.last_page}
    />
  )
}
```

---

## DATA SOURCE

- Data diambil dari backend Laravel controller melalui props halaman Inertia (termasuk pagination metadata).
- Filter pencarian dan pagination server-side dikirim dengan me-reload route menggunakan query parameters via Inertia router (`router.get`).
