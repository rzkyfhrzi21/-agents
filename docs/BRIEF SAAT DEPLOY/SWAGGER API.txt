# BRIEF MODUL: Swagger API Documentation
# Project: YUBASHOP
# Stack: Laravel 12 + Scramble (auto-generate)
# Lokasi: /docs/api (route browser)

---

## DESKRIPSI

Dokumentasi API otomatis menggunakan Laravel Scramble.
Scramble baca kode Laravel (routes, controllers, FormRequest, Resource)
dan generate OpenAPI spec TANPA annotation manual.

---

## INSTALASI

```bash
composer require dedoc/scramble
php artisan vendor:publish --provider="Dedoc\Scramble\ScrambleServiceProvider"
```

## AKSES

- Browser: https://YUBASHOP.com/docs/api
- JSON spec: https://YUBASHOP.com/docs/api.json

## KONFIGURASI (config/scramble.php)

```php
return [
    'api_path' => 'api/v1',
    'servers' => [['url' => env('APP_URL') . '/api/v1']],
    'ui' => [
        'title' => 'YUBASHOP API',
        'description' => 'API Documentation for YUBASHOP',
    ],
];
```

## AGAR DOCS BAGUS — Tulis Kode yang Clean

1. FormRequest → otomatis jadi request body schema
2. API Resource → otomatis jadi response schema
3. Return type di Controller → Scramble detect response type

## PROTEKSI DI PRODUCTION

```php
Scramble::registerUiRoute('docs/api')->middleware('auth:sanctum');
```

## CATATAN

- TIDAK PERLU annotation @OA\Get, @OA\Post
- FormRequest + API Resource yang proper = docs otomatis bagus
- Update otomatis setiap kali kode berubah
