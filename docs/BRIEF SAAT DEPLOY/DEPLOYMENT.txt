# BRIEF MODUL: Deployment Checklist
# Project: YUBASHOP
# Stack: Next.js 15 + Laravel 12 + MySQL + Redis + MinIO + Docker + Coolify
# BUKAN Vercel + Supabase — ini self-hosted via Coolify

---

## DESKRIPSI

Checklist pre-deployment untuk YUBASHOP. Deploy target: VPS via Coolify
dengan Docker Compose. Semua service (Next.js, Laravel, MySQL, Redis, MinIO)
berjalan di container Docker.

---

📋 KATEGORI 1 — NEXT.JS & KODE FRONTEND

- [ ] `npm run build` berhasil tanpa error
- [ ] Tidak ada TypeScript errors (`npx tsc --noEmit`)
- [ ] Hapus semua console.log yang tertinggal (grep -r "console.log" src/)
- [ ] Halaman 404 (not-found.tsx) dan error (error.tsx) sudah custom
- [ ] Metadata (title, description) di setiap halaman
- [ ] Environment variables: NEXT_PUBLIC_API_URL, NEXT_PUBLIC_MIDTRANS_CLIENT_KEY
- [ ] Axios instance baseURL mengarah ke env variable, bukan hardcode
- [ ] loading.tsx ada di setiap route dashboard
- [ ] Image optimization: semua gambar via next/image atau MinIO URL

---

📋 KATEGORI 2 — LARAVEL & BACKEND

- [ ] `php artisan test` → semua test passing
- [ ] `php artisan config:cache` berhasil
- [ ] `php artisan route:cache` berhasil
- [ ] Migration fresh seeder berhasil di environment bersih
- [ ] .env production: APP_DEBUG=false, APP_ENV=production
- [ ] APP_KEY sudah di-generate (`php artisan key:generate`)
- [ ] CORS dikonfigurasi hanya untuk domain frontend
- [ ] Sanctum stateful domains sesuai domain production
- [ ] Queue worker berjalan (Laravel Horizon)
- [ ] Scheduler (cron) terdaftar untuk task terjadwal

---

📋 KATEGORI 3 — DATABASE (MySQL)

- [ ] Migration kompatibel MySQL + PostgreSQL (no MySQL-only types)
- [ ] Index sudah dibuat di kolom yang sering di-filter/join:
  - transactions: trx_id, status, product_id
  - products: game_id, product_code
  - games: slug
  - users: email, role_id
- [ ] Seeder data awal: roles, config tables, superadmin user
- [ ] Backup otomatis dikonfigurasi (mysqldump cron atau Coolify backup)
- [ ] Volume MySQL di-persist di Docker volume

---

📋 KATEGORI 4 — MIDTRANS

- [ ] Ganti dari Sandbox key ke Production key di .env
- [ ] MIDTRANS_SERVER_KEY dan MIDTRANS_CLIENT_KEY di .env production
- [ ] Notification URL (webhook) diupdate ke domain production:
  https://api.YUBASHOP.com/api/v1/webhooks/midtrans
- [ ] Finish/Unfinish/Error redirect URL diupdate ke domain frontend
- [ ] Signature Key verification aktif di webhook handler
- [ ] Test 1 transaksi real di production sebelum go-live

---

📋 KATEGORI 5 — MINIO (Object Storage)

- [ ] MinIO container running di Docker Compose
- [ ] Bucket `YUBASHOP-public` dibuat dan policy: public read
- [ ] Bucket `YUBASHOP-private` dibuat dan policy: private
- [ ] MINIO_ENDPOINT, MINIO_ACCESS_KEY, MINIO_SECRET_KEY di .env
- [ ] Laravel filesystem disk 'minio' terkonfigurasi
- [ ] Volume MinIO di-persist di Docker volume
- [ ] Console MinIO (port 9001) TIDAK exposed ke publik di production

---

📋 KATEGORI 6 — DOCKER & COOLIFY

- [ ] docker-compose.yml tested: `docker compose up --build`
- [ ] Semua service healthy: next, laravel, mysql, redis, minio
- [ ] Volume persistence: mysql_data, redis_data, minio_data
- [ ] Network internal antar container (bukan expose semua port)
- [ ] Coolify deployment connected ke GitHub repository
- [ ] SSL/HTTPS aktif via Coolify (Let's Encrypt)
- [ ] Domain DNS pointing ke VPS IP

---

📋 KATEGORI 7 — KEAMANAN

- [ ] auth:sanctum di semua admin endpoint
