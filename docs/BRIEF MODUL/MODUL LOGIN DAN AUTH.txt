# BRIEF MODUL: Login dan Autentikasi
# Project: YUBASHOP
# Stack: Laravel 12 + Laravel Session + Inertia React
# Lokasi: app/Http/Controllers/AuthController.php
#          resources/js/Pages/Auth/Login.tsx

---

## 1. DESKRIPSI

Modul autentikasi menggunakan Laravel session & cookie stateful. Status login diperiksa menggunakan middleware 'auth' bawaan Laravel, dan data user login disalurkan via Inertia Shared Props.

---

## 2. FORM LOGIN (resources/js/Pages/Auth/Login.tsx)

### Requirements Fungsional:
- Form dengan input: `email` dan `password`
- Menggunakan handler Inertia React (`useForm` dari `@inertiajs/react`)
- Tombol submit dengan status "Loading..." (spinner) saat processing
- Tampilkan error validation dari Laravel (misal: "Email atau password salah.")

### Requirements Tampilan:
- Styling menggunakan Tailwind CSS + shadcn/ui
- Form card melayang di tengah layar (centered)
- Desain minimalis dengan logo YUBASHOP di bagian atas card

---

## 3. LOGIKA PROTEKSI ROUTE

Akses dilindungi sepenuhnya di backend (`routes/web.php`):

1. **Guest Area:**
   - Akses `/login` hanya diizinkan untuk user yang BELUM terautentikasi (menggunakan middleware `guest`).
   - Jika user sudah login mencoba ke `/login`, otomatis di-redirect ke `/admin` oleh Laravel.

2. **Admin Area (`/admin/*`):**
   - Wajib melalui middleware `auth` (web session).
   - Validasi role berdasarkan `role_id` di database:
     * `role_id` dengan point 1 = Superadmin
     * `role_id` dengan point 2 = Admin
     * Selain itu, jika tidak valid atau tidak memiliki role, kembalikan response unauthorized (403).

3. **Public/Guest Routes:**
   - Route storefront seperti `/`, `/{slug}`, dan `/cek-pesanan` dapat diakses bebas tanpa login.

## 4. INERTIA SHARED STATE (HandleInertiaRequests.php)

Informasi sesi dikirim ke React via middleware `HandleInertiaRequests.php`:

```php
public function share(Request $request): array
{
    return array_merge(parent::share($request), [
        'auth' => [
            'user' => $request->user() ? [
                'id' => $request->user()->id,
                'name' => $request->user()->name,
                'email' => $request->user()->email,
                'role' => $request->user()->role, // point 1 (superadmin), point 2 (admin)
            ] : null,
        ],
    ]);
}
```

Akses di file React component:
```tsx
import { usePage } from '@inertiajs/react';

const { auth } = usePage().props;
if (auth.user) {
    // User terautentikasi
}
```
