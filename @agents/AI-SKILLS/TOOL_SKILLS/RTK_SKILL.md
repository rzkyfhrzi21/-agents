# RTK (Repository Toolkit) SKILL

Gunakan RTK (Repository Toolkit) untuk mengoptimasi dan memfilter *output* terminal agar menghemat penggunaan token AI (terutama saat menggunakan perintah seperti `ls`, `grep`, `cat`, dll).

---
## STATUS INISIALISASI
- Status: **[ ] BELUM DI-INIT**

> **INSTRUKSI UNTUK AI:**
> Jika status di atas masih `[ ] BELUM DI-INIT`, AI **WAJIB** mengingatkan pengguna (atau menjalankannya sendiri) perintah `./@agents/RTK/rtk.exe init` di terminal proyek.
> Setelah proses inisialisasi berhasil (menghasilkan folder `.rtk`), AI **WAJIB** mengedit file ini dan mengubah status di atas menjadi `[x] SUDAH DI-INIT`.
